# ChatGPT用画像生成プロンプト一覧

以下のプロンプトを個別に、またはまとめてChatGPT (DALL-E 3)に入力して画像を生成してください。
生成された画像は、指定されたファイル名で `images/` フォルダに保存してください。

## 1. hero-mockup.png
**配置場所:** Hero右側
**プロンプト:**
```text
ダークな背景（#0a0f1e ダークネイビー）の上に、SaaSダッシュボードのモックアップイラストを描いてください。

画面の中に3ステップのワークフローが横並びで表示されています：
- 左: 赤い枠線のCSVファイルアイコン（エラーを示す小さな×印付き）
- 中央: オレンジ（#ff9f43）と青（#54a0ff）のグラデーションで光るAI処理ノード。歯車や波形のモチーフで「処理中」を表現
- 右: 緑の枠線の完成ドキュメントアイコン（チェックマーク付き）

3つのステップは、青（#54a0ff）のなめらかな流線で接続されています。
全体はすりガラス風（半透明＋ぼかし）のパネルの上に載っています。
背景にはごく薄いドットグリッドが散りばめられています。

スタイル: 日本のSaaS LP風のクリーンなフラットデザイン。SmartHRやfreeeの製品画面イメージに近い、柔らかく信頼感のあるトーン。
テキストは一切入れないでください。3D表現はなし。
カラーパレット: ダーク背景、白、オレンジアクセント、ブルーアクセント。
```

## 2. evolution-cycle.png
**配置場所:** Evolutionセクション
**プロンプト:**
```text
ダーク背景（#111b30）の上に、4ステップの循環サイクルインフォグラフィック図。

横一列に4つのアイコンが並び、それぞれすりガラス風カードの上に載っている:
1. ビジネスパーソンのシルエットアイコン — オレンジ（#ff9f43）の淡いグロー
2. 脳 / ニューラルネットワークのアイコン — 青（#54a0ff）の淡いグロー
3. 右肩上がりのグラフアイコン — ティール（#00d2d3）の淡いグロー
4. 地球儀アイコン（他より少し大きめ）— 青（#54a0ff）の明るいネオンボーダー

各ステップ間を、なめらかな曲線の矢印（青 #54a0ff）が繋いでいる。
4→1に戻る矢印もあり、循環を示す。背景にごく薄いドットパターン。

スタイル: 日本のSaaSでよく見る、分かりやすいインフォグラフィック調。横長（16:9）。テキストなし。
```

上部にオレンジ→青のグラデーションプログレスバー。
すりガラス風パネル。温かみのある、でもプロフェッショナルなトーン。テキストなし。

## 7-9. problem-icon-1.png / 2.png / 3.png
**配置場所:** Problems
**プロンプト (3枚個別に生成):**
```text
以下の3つのアイコン画像をそれぞれ個別に生成してください。全て正方形、ダークネイビー背景（#0a0f1e）、ミニマルな線画スタイル。

アイコン1（problem-icon-1）:
書類アイコンで中身がほぼ空白。しぼんで生気のない印象。白い線画にオレンジ（#ff9f43）のほのかなグロー。

アイコン2（problem-icon-2）:
盾のアイコンにヒビ＋ビックリマーク。アカウント停止の危険を表現。白い線画に赤〜オレンジのほのかなグロー。

アイコン3（problem-icon-3）:
ロボット（AI）の顔アイコンの周りに「？」マークと曖昧な吹き出し。的外れなAI指摘を表現。白い線画に青（#54a0ff）のほのかなグロー。
```

## 10-12. usecase-staffing.png / multistore.png / agency.png
**配置場所:** Use Cases
**プロンプト (3枚個別に生成):**
```text
以下の3つのイラストをそれぞれ個別に生成してください。全てダーク背景（#0d1528）、日本人好みのフラットイラストスタイル。

イラスト1（usecase-staffing）:
2-3人のビジネスパーソンが大量の書類を効率的にさばいている。書類がAI処理を通って整理されるイメージ。青（#54a0ff）アクセント。テキストなし。

イラスト2（usecase-multistore）:
中央に本社ビル、周囲に5-6個の店舗がネットワーク線で接続。本社から各店舗に盾（コンプライアンス保護）が行き渡るイメージ。ティール（#00d2d3）アクセント。テキストなし。

イラスト3（usecase-agency）:
パイプライン/ファネル図。左から普通の書類が入り、右からキラキラ付きの高品質ドキュメントが出てくる。星マークのクオリティバッジ付き。オレンジ（#ff9f43）アクセント。テキストなし。
```
