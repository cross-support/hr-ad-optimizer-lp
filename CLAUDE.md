クライアント: 株式会社クロスリンク様

## Tech Stack
- HTML + CSS + JavaScript（フレームワークなし）
- レスポンシブ対応（モバイルファースト）

## Rules

### 1. コミット粒度
- 作業ごとに細かく分割してコミットすること
- 1セクション = 1コミットを基本とする
- 共通スタイル・構造変更も独立したコミットにする
- コミットメッセージは日本語OK、変更内容を明確に記述

### 2. Push 前レビュー
- push 前には必ず Codex によるレビューを実施すること
- レビューで指摘された問題は修正してからコミットする
- Critical/High の指摘が残った状態で push しない

### 3. サブエージェント委任（PM スタイル）
- 基本的に実装作業はサブエージェント（task-worker 等）に委任する
- メインエージェントは PM として振る舞い、コンテキスト消費を最小化する
- 各サブエージェントには必要な情報（仕様・デザイン指示）を明示的に渡す
- サブエージェントの成果物は統合時に確認する

## Design Concept
- 信頼感（青・紺）× 先進性（テック感）× 親しみやすさ（余白多め）
- ターゲット: 派遣会社・多店舗企業・求人広告代理店の担当者
